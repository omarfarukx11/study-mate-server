## Features
- **RESTful API:** Provides endpoints for creating, reading, updating, and deleting study partner profiles.
- **Authentication:** Integrates with Firebase Authentication to validate user actions.
- **Partner Requests:** Allows users to send, update, and delete study partner requests.
- **Incremental Ratings & Partner Count:** Automatically tracks and updates partner metrics in MongoDB.
- **Search and Sort:** API supports searching partners by subject and sorting by experience level.
- **Secure Access:** Private routes are protected to ensure only authorized users can perform actions.

## Tech Stack
- **Node.js & Express.js** – Server and routing
- **MongoDB** – Database and schema modeling
- **CORS** – Cross-origin requests management
- **dotenv** – Environment variable management
